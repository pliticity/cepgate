Try the desktop version of Cepgate which enables Check-In/Check-Out functionality!

Download
Windows version

Setup

  • Open cepgate-destkop using exe file provided in the package
  • Log into desktop application (you can minimize the window after the synch screen pops out)
  • Log in to Cepgate webapp (the same account as in the desktop version)

Use

  • Choose any DIC with files attached
  • Choose "Checkout file for edit" option on the desired file
  • The file should get locked (delete and change name options should be disabled + new option "Check in file" and "Refresh status" should appear
  • The file should open on desktop
  • Desktop application detects when there are new versions of the file and asks whether to upload the file to cepgate.
  • You can continue the edit or you can check in the file to cepgate. File gets unlocked on the server.
  • Alternatively you can unlock the file manually in the webapp using check in button
  • You can hit "Refresh status" button to refresh information on file status.
